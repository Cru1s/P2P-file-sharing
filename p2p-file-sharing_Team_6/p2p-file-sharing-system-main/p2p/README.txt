- This project was built on JDK 22

- To run the server (the central peer), run file "CentralIndexingServer.java" in src/server

- To initiate the Client, run file "Client.java" in src/client

- In order to connect to the central peer, the client needs to input the full directory of the folder they want to share, client's port (to your liking), server's ip (can be check through ipconfig), and server port (default is 3434, you can change it in the code)

- The functionalities of the client has been listed inside the project.
